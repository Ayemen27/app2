## 4. تفاصيل كل شاشة

### 4.1 لوحة التحكم (Dashboard)

**الويب:**
- إحصائيات عامة
- آخر المشاريع
- آخر العمليات
- رسوم بيانية

**الموبايل MVP:**
- إحصائيات عامة (بطاقات)
- آخر 5 مشاريع
- آخر 10 عمليات
- مؤشر المزامنة

**الموبايل لاحقاً:**
- رسوم بيانية مبسطة
- إشعارات سريعة

### 4.2 المشاريع

**مشترك:**
- قائمة المشاريع مع البحث والتصفية
- عرض التفاصيل
- إضافة/تعديل/حذف

**إضافات الموبايل:**
- GPS لتحديد الموقع
- تصوير صور للمشروع
- مؤشر حالة المزامنة لكل مشروع

### 4.3 العمال

**مشترك:**
- قائمة العمال مع البحث
- تفاصيل العامل
- إضافة/تعديل/حذف
- ربط بمشروع

**إضافات الموبايل:**
- صورة العامل من الكاميرا
- الاتصال المباشر (tap to call)

### 4.4 الحضور

**مشترك:**
- تسجيل الحضور اليومي
- عرض سجل الحضور
- حساب المستحقات

**إضافات الموبايل:**
- GPS timestamp
- العمل Offline الكامل
- مزامنة ذكية للحضور

### 4.5 شراء المواد

**مشترك:**
- تسجيل عملية الشراء
- ربط بالمورد
- حساب الإجمالي

**إضافات الموبايل:**
- تصوير الفاتورة
- ضغط الصور تلقائياً
- رفع مؤجل عند الاتصال
