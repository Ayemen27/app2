# توثيق تكامل الأندرويد (Android Integration Documentation)

## 1. التزامن الذكي (Smart Sync)
تم إنشاء موديول `mobile/sync.ts` ليكون المسؤول الوحيد عن تزويد تطبيق الأندرويد بالبيانات.
- **تحديثات احترافية**: النظام الآن يدعم التزامن الجزئي والكلي مع تقليل استهلاك البيانات.
- **التوافق**: تم الحفاظ على هيكل البيانات القديم لضمان عدم توقف إصدارات التطبيق الحالية.

## 2. المسارات الموحدة (Unified Endpoints)
- `GET /api/mobile/sync/full-backup`: للحصول على نسخة كاملة من البيانات.
- `POST /api/mobile/auth/login`: نقطة دخول موحدة للهاتف والويب.

## 3. معايير الأداء (Performance Metrics)
تم إضافة `metadata` لكل استجابة برمجية لمراقبة أداء السيرفر وسرعة معالجة طلبات الهاتف.
