## 5. النماذج المدعومة | Supported Models

### النموذج الأساسي: OpenAI
```typescript
{
  provider: "openai",
  model: "gpt-5",
  apiKey: process.env.OPENAI_API_KEY,
  priority: 1,
  dailyLimit: 1000 // requests
}
```

### النموذج الاحتياطي: Google Gemini
```typescript
{
  provider: "gemini",
  model: "gemini-2.5-flash",
  apiKey: process.env.GEMINI_API_KEY,
  priority: 2,
  dailyLimit: 1500 // requests (مجاني أكثر)
}
```

### آلية التبديل:
1. استخدام OpenAI أولاً
2. عند الخطأ (429 Rate Limit) → التبديل إلى Gemini
3. عند استعادة OpenAI → العودة للاستخدام
4. تتبع الاستخدام اليومي لكل نموذج
