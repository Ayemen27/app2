## 5. النماذج المدعومة | Supported Models

### النموذج الأساسي: OpenAI
```typescript
{
  provider: "openai",
  model: "gpt-4o",
  apiKey: process.env.OPENAI_API_KEY,
  priority: 1,
  dailyLimit: 1000
}
```

### النموذج الاحتياطي: Google Gemini
```typescript
{
  provider: "gemini",
  model: "gemini-2.0-flash",
  apiKey: process.env.GEMINI_API_KEY || process.env.GOOGLE_API_KEY,
  priority: 2,
  dailyLimit: 1500
}
```

### آلية التبديل:
1. استخدام OpenAI أولاً
2. عند الخطأ (429 Rate Limit) أو تجاوز الحد اليومي → التبديل إلى Gemini
3. إعادة تعيين العدادات يومياً
4. تتبع الاستخدام اليومي لكل نموذج
