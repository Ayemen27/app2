# مشروع نظام مراقبة أندرويد المتكامل

## نظرة عامة
نظام احترافي لمراقبة تطبيقات أندرويد يجمع بين تتبع الأداء (OTel)، إدارة الأخطاء (Sentry)، ولوحة تحكم متطورة (Next.js).

## هيكل المشروع
- `system_core/docs/`: يحتوي على خطط العمل والمعايير.
- `client/`: (مستقبلي) لوحة التحكم الإدارية.
- `server/`: (مستقبلي) API Backend و Collector.

## تفضيلات العمل
- استخدام **Fast Mode** لتنفيذ المهام الذرية.
- الالتزام بمعايير **OpenTelemetry**.
- التصميم باستخدام **Tailwind + Shadcn**.
