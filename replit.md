# AXION Operations Management | أكسيون لإدارة العمليات

## Overview
AXION is a sophisticated management system for oil well operations, financial reporting, and real-time monitoring. It features an integrated AI agent and a dual-language (Arabic/English) interface.

## Technical Stack
- **Backend:** Express.js with Drizzle ORM (PostgreSQL).
- **Frontend:** React, Tailwind CSS, Shadcn UI.
- **Monitoring:** SigNoz (OTEL), Sentry.
- **AI:** OpenAI/ModelManager integrated.

## Project Structure
- `client/`: Frontend React application.
- `server/`: Express backend and API routes.
- `shared/`: Shared schemas and types.
- `system_core/`: Documentation and global standards.
- `www/`: Static assets and production build.
