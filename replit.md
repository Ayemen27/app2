# مشروع إدارة العمليات المتكامل

## نظرة عامة
نظام متطور لإدارة عمليات الآبار، التقارير المالية، والمراقبة اللحظية، مدعوم بوكيل ذكاء اصطناعي داخلي.

## بروتوكول العمل (قواعد صارمة)
- **الدقة فوق السرعة:** لا يتم تقديم حلول غير مؤكدة.
- **التحليل المنطقي:** التفكير خطوة بخطوة قبل كل إجابة.
- **الواقعية:** اختيار الحلول القابلة للتنفيذ وتقييم التكلفة/السرعة.
- **الشفافية:** التصريح الصريح عند نقص المعلومات أو عدم المعرفة.
- **منع المجاملة:** تقديم الواقع التقني كما هو.

## الهيكل التقني
- **Backend:** Express.js مع Drizzle ORM.
- **Frontend:** React + Tailwind CSS + Shadcn UI.
- **Services:** نظام مراقبة (Signoz), إدارة أسرار, ووكيل ذكاء اصطناعي (OpenAI/ModelManager).
