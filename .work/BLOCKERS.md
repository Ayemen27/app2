## 📋 التنسيق

لكل مشكلة:
```markdown
### مشكلة #X: [الاسم]
**الوصف**: ماذا حدثت المشكلة
**التأثير**: أي مهام / مراحل تتأثر
**الخطورة**: 🔴 حرجة / 🟡 عالية / 🟢 منخفضة
**الحالة**: ⏳ معلقة / 🔄 قيد الحل / ✅ مُحلّة
**الحل المقترح**: كيفية الحل
**الملاحظات**: معلومات إضافية
```

## 📝 نموذج مشكلة جديدة

اضغط + وأضف:

```markdown
### مشكلة #[عدد متسلسل]: [عنوان المشكلة]

**الوصف**: 
[شرح تفصيلي للمشكلة]

**الخطوات لإعادة الإنتاج**:
1. ...
2. ...
3. ...

**السلوك المتوقع**:
[ماذا يجب أن يحدث]

**السلوك الفعلي**:
[ماذا يحدث بالفعل]

**التأثير**:
- ❌ المهام المتأثرة: (مثل 1.1, 2.3)
- ⏸️ المراحل المعلقة: (مثل Phase 2, Phase 3)

**الخطورة**: 🔴 🟡 🟢

**الحالة**: ⏳ معلقة

**الحل المقترح**:
[خيارات محتملة]

**محاولات سابقة**:
[أي حلول تم تجربتها]

**من اكتشفها**: [اسم الوكيل]

**التاريخ**: [التاريخ]

**ملاحظات إضافية**:
[أي معلومات أخرى مفيدة]
```

## 🔄 تحديث المشاكل

### عندما تتقدم المشكلة
1. حدّث `الحالة` من ⏳ إلى 🔄
2. أضف تعليق بآخر التطورات
3. وثّق أي معلومات جديدة

### عندما تُحل المشكلة
1. غيّر `الحالة` إلى ✅
2. اشرح الحل النهائي
3. أضف رقم commit إن وجد
4. انقل المشكلة إلى قسم "المشاكل المحلولة"
